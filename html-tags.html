<a> Tag

The <a> tag defines a hyperlink, which is used to link to another page or location within the same page.

<a href="https://example.com">Visit Example</a>

<img> Tag

The <img> tag is used to embed images into the webpage. It requires the 'src' attribute for the image source.

<img src="image.jpg" alt="Description of the image">

<p> Tag

The <p> tag defines a paragraph of text.

<p>This is a paragraph.</p>

<h1> to <h6> Tags

These tags define headings ranging from the largest (<h1>) to the smallest (<h6>).

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Sub-subheading</h3>

<div> Tag

The <div> tag is a container for grouping elements, often styled with CSS or manipulated with JavaScript.

<div>
    <p>This is inside a div.</p>
</div>

<ul> and <li> Tags

These tags create unordered lists (<ul>) with list items (<li>), commonly used for bullet points.

<ul>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ul>

<table> Tag

The <table> tag defines a table in HTML to display data in rows and columns.

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Row 1, Col 1</td>
        <td>Row 1, Col 2</td>
    </tr>
</table>

<form> Tag

The <form> tag defines an HTML form for user input, used to collect data from users.

<form action="submit.php" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <input type="submit" value="Submit">
</form>

<span> Tag

The <span> tag is an inline container used to mark up a part of text or a document.

<span>This is inline content.</span>

<header> Tag

The <header> tag represents a container for introductory or navigational aids. It typically includes the site title or logo and navigation links.

<header>
    <h1>Site Title</h1>
    <nav>...</nav>
</header>

<footer> Tag

The <footer> tag defines a footer for a document or section, often containing information like copyright details.

<footer>
    © 2025 My Website
</footer>

<nav> Tag

The <nav> tag defines a section of navigation links.

<nav>
    <ul>
        <li><a href="index.html">Home</a></li>
        <li><a href="about.html">About</a></li>
    </ul>
</nav>

<article> Tag

The <article> tag specifies independent, self-contained content, such as a blog post or news article.

<article>
    <h2>Article Title</h2>
    <p>Article content goes here...</p>
</article>

<section> Tag

The <section> tag defines a section in a document, such as chapters, headers, or any thematic grouping of content.

<section>
    <h2>Section Title</h2>
    <p>Section content goes here...</p>
</section>

<aside> Tag

The <aside> tag defines content aside from the page's main content, often used for sidebars or call-out boxes.

<aside>
    <p>Side content...</p>
</aside>

<script> Tag

The <script> tag is used to embed or reference JavaScript code in an HTML document.

<script>
    console.log('Hello, world!');
</script>

<link> Tag

The <link> tag specifies the relationship between the current document and an external resource. It is most commonly used to link CSS stylesheets.

<link rel="stylesheet" href="styles.css">

<meta> Tag

The <meta> tag defines metadata about an HTML document such as its character encoding, description, or keywords. It is always placed inside the head of the document.

<meta charset="UTF-8">

<iframe> Tag

The <iframe> tag allows you to embed another HTML document within the current page.

<iframe src="https://example.com" title="Example"></iframe>

<em> Tag

The <em> tag is used to emphasize text, usually rendered in italics.

<em>This text is emphasized.</em>

<strong> Tag

The <strong> tag indicates that its content is of strong importance, typically rendered in bold.

<strong>Important text</strong>
