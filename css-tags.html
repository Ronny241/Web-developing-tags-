CSS Selectors

Selectors target HTML elements to apply styles. Examples include element, class, and ID selectors:

/* Element Selector */
h1 {
    color: blue;
}

/* Class Selector */
.box {
    background-color: lightblue;
}

/* ID Selector */
#header {
    text-align: center;
}

CSS Box Model

The box model includes content, padding, borders, and margins:

div {
    padding: 10px;
    border: 2px solid #00796b;
    margin: 15px;
    width: 200px; /* Content Width */
    height: 100px; /* Content Height */
}

Typography

Control the appearance of text using typography properties:

p {
    font-family: 'Roboto', sans-serif;
    font-size: 18px;
    color: #333;
    text-align: justify;
    line-height: 1.8;
}

Colors and Backgrounds

CSS allows for vibrant styles using colors and background properties.

body {
    background-color: #e0f7fa; /* Solid Color */
}

h1 {
    color: #006064; /* Text Color */
}

div {
    background: linear-gradient(to right, #0288d1, #004d40); /* Gradient */
}

CSS Animations

Use CSS animations to add movement and effects:

@keyframes bounce {
    0%, 100% {
        transform: translateY(0);
    }
    50% {
        transform: translateY(-30px);
    }
}

div {
    animation: bounce 2s infinite;
}

CSS Positioning

CSS positioning controls the placement of elements:

/* Relative Position */
div {
    position: relative;
    top: 10px;
    left: 20px;
}

/* Absolute Position */
div {
    position: absolute;
    top: 50px;
    left: 50px;
}

CSS Flexbox

Flexbox is used for aligning elements within a container:

.container {
    display: flex;
    justify-content: center;
    align-items: center;
}

CSS Grid

Grid layout provides a two-dimensional layout system for the web:

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
}
