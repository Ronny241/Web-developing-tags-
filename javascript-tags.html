JavaScript Variables

Variables store data that can be used later:

// Declaring variables
let name = "John";
const age = 25;
console.log(name, age); // Outputs: John 25

JavaScript Functions

Functions allow you to reuse blocks of code:

// Defining a function
function greet(name) {
    return `Hello, ${name}!`;
}

// Using the function
console.log(greet("Alice")); // Outputs: Hello, Alice!

JavaScript Events

Events allow you to handle user interactions:

// Adding an event listener
document.querySelector("button").addEventListener("click", function() {
    alert("Button clicked!");
});

JavaScript Loops

Loops allow you to repeat blocks of code:

// Using a for loop
for (let i = 0; i < 5; i++) {
    console.log(i);
}

DOM Manipulation

JavaScript can dynamically modify the DOM:

// Changing text content
document.querySelector("h1").textContent = "JavaScript is Fun!";

ES6 Features

Modern JavaScript (ES6) includes new features like let/const and arrow functions:

// Arrow function
const add = (a, b) => a + b;

console.log(add(5, 3)); // Outputs: 8

JavaScript Objects

Objects store collections of data and methods:

const person = {
    name: "Alice",
    age: 30,
    greet() {
        return `Hi, I'm ${this.name}`;
    }
};

console.log(person.greet()); // Outputs: Hi, I'm Alice
