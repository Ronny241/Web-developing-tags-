Welcome to CSS

Explore examples of common CSS properties and concepts below.

1. Selectors

CSS selectors are used to select and style specific HTML elements. Common types include:

  • Element Selector: Targets an HTML element, like h1.
  • Class Selector: Targets elements with a specific class, like .animation.
  • ID Selector: Targets an element with a specific ID, like #header.
/* Element Selector */
h1 {
    color: blue;
}

/* Class Selector */
.animation {
    background-color: green;
}

/* ID Selector */
#header {
    background-color: yellow;
}

2. Colors and Backgrounds

CSS allows you to add colors and backgrounds to elements.

/* Text Color */
h1 {
    color: #00796b;
}

/* Background Color */
body {
    background-color: #e0f7fa;
}

/* Gradient Background */
div {
    background: linear-gradient(to right, #004d40, #00796b);
}

3. Box Model

The CSS box model consists of margins, borders, padding, and the content area.

div {
    margin: 20px; /* Space outside the element */
    border: 2px solid #004d40; /* Border around the element */
    padding: 10px; /* Space inside the border */
    width: 100px; /* Width of the content area */
}

4. Fonts and Text

CSS provides properties to style fonts and text.

body {
    font-family: Arial, sans-serif; /* Font style */
    font-size: 16px; /* Font size */
    color: #333; /* Text color */
    text-align: center; /* Align text */
}

5. Animations

CSS animations let you create dynamic visual effects.

@keyframes bounce {
    0%, 100% {
        transform: translateY(0);
    }
    50% {
        transform: translateY(-50px);
    }
}

div {
    animation: bounce 2s infinite; /* Apply the animation */
}

6. Positioning

CSS positioning allows you to control the layout of elements on the page.

/* Static (default) */
div {
    position: static;
}

/* Relative */
div {
    position: relative;
    top: 10px;
    left: 10px;
}

/* Absolute */
div {
    position: absolute;
    top: 50px;
    left: 50px;
}

/* Fixed */
div {
    position: fixed;
    bottom: 10px;
    right: 10px;
}
