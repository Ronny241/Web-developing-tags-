Welcome to Learn HTML

Explore examples of common HTML tags below.

1. The <html> Tag

The <html> tag represents the root of an HTML document. All content of the page is enclosed within this tag.

<html>
    <!-- Content here -->
</html>

2. The <head> Tag

The <head> tag contains metadata about the HTML document, like the title, character encoding, and linked resources.

<head>
    <title>My Website</title>
    <meta charset="UTF-8">
</head>

3. The <body> Tag

The <body> tag contains the visible content of the web page, such as text, images, and interactive elements.

<body>
    <h1>Welcome to My Website</h1>
    <p>This is a sample paragraph.</p>
</body>

4. The <h1> to <h6> Tags

These tags define headings, from the largest (<h1>) to the smallest (<h6>).

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Sub-subheading</h3>

5. The <p> Tag

The <p> tag defines a paragraph of text.

<p>This is a paragraph.</p>

6. The <a> Tag

The <a> tag defines a hyperlink, which links to another web page or a location within the same page.

<a href="https://example.com">Visit Example</a>

7. The <div> Tag

The <div> tag is a container for grouping elements, often styled with CSS or manipulated with JavaScript.

<div>
    <p>This is inside a div.</p>
</div>

8. The <img> Tag

The <img> tag embeds an image in the page. It requires the src attribute to specify the image source and the alt attribute for accessibility.

<img src="image.jpg" alt="Description of the image">
