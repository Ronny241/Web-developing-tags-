Welcome to JavaScript

Learn the basics of JavaScript through interactive examples below.

1. Variables

JavaScript variables store data that can be used later. Examples:

// Declaring variables
let name = "John";
const age = 25;
console.log(name, age); // Outputs: John 25

2. Functions

Functions allow you to reuse blocks of code:

// Defining a function
function greet(name) {
    return `Hello, ${name}!`;
}

// Using the function
console.log(greet("Alice")); // Outputs: Hello, Alice!

Try it:

3. Events

JavaScript allows you to handle user interactions using events:

// Adding an event listener
document.querySelector("button").addEventListener("click", function() {
    alert("Button clicked!");
});

Click the button above to see an example!

4. Loops

Loops are used to repeat a block of code:

// Using a for loop
for (let i = 0; i < 5; i++) {
    console.log(i);
}

5. DOM Manipulation

JavaScript can dynamically change the content of your webpage:

// Changing text content
document.querySelector("h1").textContent = "JavaScript is Fun!";
